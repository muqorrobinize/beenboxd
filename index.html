Panduan Penggunaan Lengkap

1. Pengenalan

BeenBoxd adalah tool pintar untuk mengonversi catatan film yang "berantakan" menjadi file CSV yang rapi dan kompatibel dengan sistem Import Letterboxd.

Kami menggunakan Smart Queue System (Antrean Pintar) untuk memecah daftar ribuan film menjadi potongan kecil, sehingga prosesnya stabil dan tidak membebani server.

FITUR UTAMA AI Bulk Processing

Cara paling efisien untuk memindahkan data dari Notes/Excel/WhatsApp ke Letterboxd.

Langkah-Langkah:

  1. Klik tab AI BULK di panel kiri.
  2. Tempel (Paste) daftar film Anda. Format tidak harus rapi.
  3. (Opsional) Tulis instruksi tambahan di baris yang sama atau di bawahnya.
  4. Klik tombol PROCESS QUEUE.
  5. Tunggu hingga Progress Bar mencapai 100%.

Contoh Input yang Valid:

Oppenheimer
Barbie (2023)
The Bear Season 1
Succession S4

-- Instruksi: Kasih rating 5 buat Succession, sisanya 4. Tag 'binge-watch'.

2. Input Manual

Gunakan ini untuk menambah satu atau dua film secara spesifik.

  • Ketik judul di kolom Title.
  • Klik tombol AUTO di sebelahnya.
  • AI akan otomatis mencari Tahun Rilis yang benar dan mengisi kolom Tags dengan genre film tersebut.
  • Lengkapi Rating, Tanggal, dan Review sesuai keinginan.

3. Ekspor & Upload

  1. Setelah semua data masuk ke tabel kanan, klik tombol EXPORT CSV.
  2. File bernama letterboxd-import-[TANGGAL].csv akan terunduh.
  3. Pergi ke Halaman Import Letterboxd.
  4. Upload file tersebut. Selesai!

Troubleshooting / Masalah Umum

  • Progress Bar Macet? Tunggu sebentar. Sistem sedang melakukan "retry" otomatis jika server AI sedang sibuk.
  • Error Merah? Coba kurangi jumlah baris input (misal per 50 judul) jika input sangat panjang.
  • Data Hilang saat Refresh? Tidak perlu khawatir. Data disimpan di browser (LocalStorage) secara otomatis.
